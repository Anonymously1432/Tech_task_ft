# Название проекта

Сервис для оформления пользователям различных страховок. Состоит из клиентской и менеджерской частей (SPA) и основного бэкенд-сервера.

## Архитектура

Проект построен на микросервисной архитектуре и включает в себя:

- **Клиентское SPA** - пользовательский интерфейс (http://localhost:3000)
- **Менеджерское SPA** - интерфейс для администраторов/менеджеров (http://localhost:3001)
- **Основной бэкенд** - API сервер (http://localhost:8080/api/v1/docs/index.html)

### Технологический стек

- **Бэкенд**: Go (Golang)
- **Фронтенд**: React.js
- **Контейнеризация**: Docker, Docker Compose

## Быстрый старт

### Предварительные требования

- Установленные [Docker](https://docs.docker.com/get-docker/) и [Docker Compose](https://docs.docker.com/compose/install/)
- Порты 3000, 3001, 8080 должны быть свободны

### Запуск проекта

1. Клонируйте репозиторий:
```bash
    git clone <url-репозитория>
    cd <название-проекта>
```
2. Запустите все сервисы командой
```bash
    docker compose up --build
```
3. После успешного запуска будут доступны:
```bash
    Клиентская часть: http://localhost:3000
    Менеджерская часть: http://localhost:3001
    API документация: http://localhost:8080/api/v1/docs/index.html
```
4. Для полной очистки (включая тома с данными):
```bash
    docker compose down -v
```

Сервис состоит из трех приложений - клиентское SPA, менеджерское SPA и основной бекенд
Запускается из корня командой docker compose up --build
Документация доступна по адресу http://localhost:8080/api/v1/docs/index.html
Клиентская часть достуна по адресу http://localhost:3000
Менеджерская часть достуна по адресу http://localhost:3001

Проект написан с помощью go и react
